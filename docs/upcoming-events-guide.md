# 活動輪播管理指南

管理首頁「活動輪播」區塊的活動內容。所有操作都在 Google Sheet 上完成，不需要改程式碼。

## 開啟試算表

[HOA 報名資料試算表](https://docs.google.com/spreadsheets/d/1am2e_RU_fkx--338b7F76NjjP8CM5O1wnKYJmDRubhM) → 切換到 **「UpcomingEvents」** 分頁

## 欄位說明

| 欄位 | 必填 | 說明 | 範例 |
|------|------|------|------|
| `id` | 是 | 唯一識別碼 | `event-003` |
| `title` | 是 | 活動名稱 | `AI 素養起步走｜學生體驗場` |
| `description` | 是 | 簡短描述（顯示在卡片上） | `由均一老師全程線上帶領，讓孩子趣味體驗 AI` |
| `url` | 是 | 報名連結（完整網址） | `https://link.junyiacademy.org/xxxxx` |
| `startDate` | 是 | 活動開始日期 | `2026-03-15` |
| `endDate` | 否 | 活動結束日期（留空 = 單日活動） | `2026-03-20` |
| `isActive` | 是 | 是否顯示在網頁上 | `TRUE` 或 `FALSE` |
| `sortOrder` | 否 | 排序順序，數字越小越前面 | `1` |

## 新增活動

1. 在最下方新增一列
2. 填入所有必填欄位
3. `isActive` 設為 `TRUE`
4. 等待最多 **35 分鐘**自動生效（伺服器快取 5 分鐘 + 前端快取 30 分鐘）

> 想立即看到效果？開啟瀏覽器無痕模式造訪網站即可跳過前端快取。

## 編輯活動

直接在試算表上修改對應欄位即可。常見情境：

- **改標題或描述** → 直接編輯 `title` / `description` 欄位
- **換報名連結** → 修改 `url` 欄位（務必貼完整網址）
- **調整排序** → 修改 `sortOrder` 數字

## 下架（封存）活動

將該列的 `isActive` 改為 `FALSE`。

活動資料仍保留在試算表中，方便日後查閱或重新上架。

> 不建議直接刪除列，保留歷史紀錄比較好管理。

## 活動狀態徽章

網頁會根據日期自動顯示狀態，不需要手動設定：

| 狀態 | 條件 | 顯示樣式 |
|------|------|----------|
| 即將舉行 | `startDate` 在未來 | 正常顯示 |
| 進行中 | 今天介於 `startDate` ~ `endDate` 之間 | 正常顯示 |
| 已舉行 | `endDate`（或 `startDate`）已過 | 降飽和度 |

## 注意事項

- `url` 必須是完整網址（以 `https://` 開頭），否則前端驗證會過濾掉
- `id` 不可重複
- 日期格式建議用 `YYYY-MM-DD`（如 `2026-03-15`），Google Sheet 的日期格式也可以
- 更新後不需要重新部署，快取過期後自動生效
